Posted on 2015-12-21 | Edited on 2019-01-19 | In SharePoint Add-in

I’m learning how to upgrade SharePoint App, and I met a problem when I added a new list field in the upgrade.

The document.aspx) mentioned that if you added a field to a content type in the feature, you should add an AddContentTypeField element to the VersionRange section.
